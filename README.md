# Система мониторинга и анализа температуры

Приложение Streamlit для анализа исторических данных о температуре и мониторинга текущих погодных условий с использованием OpenWeatherMap API.

## Особенности

### Анализ данных
- **Анализ временных рядов**: Интерактивные графики со скользящим средним (30 дней) и доверительными интервалами
- **Обнаружение аномалий**: Автоматическое выявление температурных аномалий с использованием статистических методов
- **Сезонные профили**: Анализ температурных паттернов по сезонам
- **Анализ трендов**: Детекция долгосрочных температурных трендов с помощью линейной регрессии
- **Параллельная обработка**: Сравнение производительности последовательной и параллельной обработки данных

### Мониторинг в реальном времени
- **Интеграция с OpenWeatherMap**: Получение текущей температуры для любого города
- **Обнаружение аномалий**: Сравнение текущей температуры с историческими данными
- **Сравнение подходов**: Сравнение производительности синхронных и асинхронных API запросов

### Интерактивная панель управления
- **Выбор города**: Выбор из доступных городов в наборе данных
- **Фильтрация по датам**: Возможность фокусировки на конкретных временных периодах
- **Множественные визуализации**: Временные ряды, сезонные профили, графики распределения и анализ трендов
- **Метрики производительности**: Сравнение параллельной обработки и асинхронных API запросов
